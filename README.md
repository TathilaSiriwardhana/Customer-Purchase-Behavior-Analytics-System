# Customer Purchase Behavior Analytics System

## Project Overview
This project analyzes customer purchase behavior using demographic data to identify key customer segments and support business decision-making.

## Dataset
- Customer demographic and purchasing data
- Features include age, gender, salary, and purchase status

## Project Structure
- `data/raw/` – Original dataset
- `data/processed/` – Cleaned and transformed datasets
- `notebooks/01_data_understanding.ipynb`
- `notebooks/02_data_cleaning.ipynb`
- `notebooks/03_exploratory_analysis.ipynb`

## Key Insights
- Customers aged **26–45** show the strongest purchasing behavior.
- Younger customers (18–25) show lower engagement.
- Older customers (46–60) remain an important segment.

## Business Recommendations
- Focus marketing efforts on 26–45 age group
- Create retention campaigns for older customers
- Re-evaluate strategies for younger customers
